# Markdown to HTML Converter CLI Tool

## 📌 Description
This is a Node.js command-line tool that converts Markdown files (`.md`) into clean and styled HTML files.

## 🚀 Features
- Headers (`#`, `##`, etc.)
- Bold (`**bold**`)
- Italics (`*italic*`)
- Lists (`- item`)
- Links (`[OpenAI](https://openai.com)`)
- Inline Code and Code Blocks
